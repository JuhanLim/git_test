**시작 방법:**
```bash
# 1. 의존성 설치
pip install -r requirements.txt

# 2. 환경변수 설정
cp .env.example .env  # SECRET_KEY 등 입력

# 3. DB 초기화 & 실행
python manage.py migrate
python manage.py runserver

# 4. (선택) 백그라운드 데이터 수집
celery -A config worker -l info
celery -A config beat -l info
```
